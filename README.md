# Quickplay Studios - Cake Game
- Play as a little cake guy trying to survive a bullet hell of forks and knives!
- Made with pygame community edition

How to download:
1. Install / check that you have the dependencies
- Make sure you have [git installed](https://git-scm.com/install/)
- Make sure you have [the latest version of python ](https://www.python.org/downloads/)
    You can check using the ```python --version``` command
- Install pygame community edition
    You can use the ```pip install pygame-ce``` command [(more details here)](https://pypi.org/project/pygame-ce/)

2. Clone the repository with ```git clone https://github.com/Jaylan-Oliver/spring26-game-jam-QuickplayStudios```
3. Run the game from the game.py file.
